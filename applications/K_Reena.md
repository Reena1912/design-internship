## 4. A project you've shipped recently

https://problem-recommender.onrender.com/

So, I built this app that basically acts as a personal coach for your leetcode grind.

You just drop in a profile, and it uses this custom scoring algorithm to figure out exactly which tags you're weak in, then ranks the problems you should tackle next.

I built the whole thing from scratch - the scraper, the backend, the dashboard everything. It started as a simple script for me, but i ended up scaling it into a proper web app that handles multiple users.

if i had more time, i’d swap the manual login for oauth so it’s smoother, and move the caching to redis so it doesn't reset every time the server restarts. it’s been a fun project to see come to life.
